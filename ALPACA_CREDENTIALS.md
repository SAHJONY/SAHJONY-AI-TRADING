# Alpaca Credentials

This repository uses Alpaca API keys which are stored in the Hermes environment at `~/.hermes/.env`. Do not commit any secrets to the repo. The required variables are:

```
APCA_API_KEY_ID=YOUR_KEY_ID
APCA_API_SECRET_KEY=YOUR_SECRET_KEY
```

Ensure these are set in the deployment environment (Vercel, etc.).
